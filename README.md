# MOOVEBIKE

Aqui estamos desenvolvendo nosso projeto integrador


## Tecnologias
 - [Express](https://expressjs.com/pt-br/)
 - [Ejs](https://ejs.co/)

## Como utilizar

Clonar o projeto e depois instalar os pacotes com npm

```bash
$ npm install
```

## Rodando local

```bash
$ npm run dev
```
